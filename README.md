# Simple College Website Layout

A basic, responsive front-end layout for a college website built using HTML, CSS, and Vanilla JavaScript. This project demonstrates a single-page application feel by toggling between sections without reloading the browser.

## 🚀 Features

* **Dynamic Navigation:** Switch between the "Home" and "Login" sections dynamically using JavaScript.
* **Moving Header:** A welcome message continuously scrolls from right to left at the top of the page.
* **Dual Login Portal:** The login section includes a toggle to switch between "Student Login" and "Staff Login" forms.
* **Custom Background:** The home page features a background image with a low-opacity overlay to ensure text remains readable.

## 🛠️ Technologies Used

* **HTML5:** For the basic structure and layout.
* **CSS3:** For styling, Flexbox layout, and background gradients.
* **JavaScript:** For DOM manipulation and handling button clicks to show/hide specific sections.
